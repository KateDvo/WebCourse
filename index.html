    • return all elements as collection
    • for searching nested elements with class name:

      create variable that contains parent element

      use 'varname'.getElementByClassName('class-name')

  • querySelector('li') / querySelectorAll('li')
    • A newer all-in-one method to select a SINGLE ELEMENT
    • querySelector('li') return only first item
    • querySelectorAll('li') return NodeList with all elements
    • for searching by id need to add #, like in css
    • for searching nested elements provide hierarchy: document.querySelector('body ul li.special')
